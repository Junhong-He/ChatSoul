# ChatSoul 使用说明（新手必读）

一句话看懂：这是一个**完全在你自己电脑上运行**的 AI 角色扮演聊天机器人。
它**不联网、不花任何钱、不发任何消息到外面**，你和角色的每一句对话都只存在你自己的电脑里。
唯一需要先做的事：装一个免费的小软件（Ollama）和一个免费的 AI 模型（约 2GB）。
第一次装好之后，以后双击就能用。

---

## 一、你要准备什么

| 需要 | 是什么 | 花钱吗 | 大概耗时 |
|------|--------|--------|----------|
| ① 这个文件夹（ChatSoul） | 已经给你了 | 免费 | — |
| ② Ollama | 一个免费的小软件，负责在本地跑 AI | 免费 | 几分钟 |
| ③ Qwen2.5:3B 模型 | AI 的"大脑"，约 2GB | 免费 | 第一次要等一会儿（几分钟到几十分钟，看网速） |

模型文件较大，**第一次必须下载一次**。之后就永久留在你电脑上，断网也能用。

---

## 二、方式一：一键安装（最推荐，基本不用动脑）

这个文件夹里有一个 `install.bat`，它可以**自动**帮你做完下面所有事：
装 Ollama → 下载模型 → 启动 ChatSoul。

操作步骤：
1. 打开这个 `ChatSoul` 文件夹。
2. **双击 `install.bat`**（不是 ChatSoul.exe，是 install.bat）。
3. 会弹出一个黑色窗口，按提示自动进行，中途可能需要联网下载，请耐心等几分钟。
4. 装完后浏览器会自动打开聊天界面，就可以开始聊了。

如果一键安装中途失败（比如网络问题），黑色窗口会告诉你原因，并提示你改用下面的"方式二"手动安装。

---

## 三、方式二：手动安装（分步）

如果一键安装不成功，或者你想自己搞清楚每一步，按下面来。

### 第 1 步：安装 Ollama（只需一次）

1. 打开浏览器，访问 https://ollama.com/download
2. 点 Windows 下载 `OllamaSetup.exe`，下载完双击安装，一路点"下一步 / Install"。
3. 安装完成后，验证一下：
   - 按键盘 `Win` + `R`，输入 `powershell`，回车（弹出蓝色或黑色窗口）；
   - 在里面输入下面这行，回车：

   ```
   ollama --version
   ```

   - 能看到类似 `ollama version 0.xx.x` 就成功了。
   - 如果提示"不是内部或外部命令"，请重开一个 PowerShell 再试；还不行就重启电脑。

### 第 2 步：把 AI 模型请到本地（二选一，只需一次）

模型就是 AI 的"大脑"。下面两种方法任选其一，装好之后就不需要再管了。

#### 方法一：最简单（推荐先试这个）

在刚才的 PowerShell 窗口里，直接复制粘贴下面这行，回车：

```
ollama pull qwen2.5:3b
```

- 它会开始下载，进度条一直走。**等它显示 success 或自动回到输入光标**就成功了。
- 模型约 2GB，网速正常几分钟到十几分钟；**国内如果下得很慢（几十 KB/s）甚至卡住，请改用"方法二"**。

#### 方法二：国内加速下载（快，但需要先装 Python）

如果你在中国大陆，且方法一下不动 / 极慢，用国内镜像会快很多。

① 先装 Python（如果已经有了就跳过）
- 打开 https://www.python.org/downloads/
- 下载 Windows 安装包，双击安装。**关键一步：安装界面最下面一定要勾选 `Add Python to PATH`**，然后点 Install。
- 装完重开一个 PowerShell，输入 `python --version` 能看到版本号即成功。

② 在 PowerShell 里，一行一行复制粘贴下面每一行（每行回车执行）：

```
python -m pip install modelscope
modelscope download --model Qwen/Qwen2.5-3B-Instruct-GGUF qwen2.5-3b-instruct-q4_k_m.gguf --local_dir ./models
echo FROM ./models/qwen2.5-3b-instruct-q4_k_m.gguf > Modelfile
echo PARAMETER num_ctx 8192 >> Modelfile
ollama create qwen2.5:3b -f Modelfile
```

③ 验证：输入下面这行，能看到 `qwen2.5:3b` 这一行就成功了：

```
ollama list
```

> 小提示：以上命令在你电脑上任意文件夹里执行都行，模型会注册到 Ollama 里，和 ChatSoul 放在哪无关。

## 四、运行 ChatSoul（以后每次都这样）

1. 打开你收到的 `ChatSoul` 文件夹（放着 `ChatSoul.exe` 的这个文件夹）。
2. **双击 `ChatSoul.exe`**（或双击 `start.bat` 也行）。
3. 程序会**自动打开你的默认浏览器**，进入聊天界面（地址一般是 `http://localhost:8000`）。
   - 万一浏览器没自动弹出来，也别慌：自己打开浏览器，在地址栏输入 `http://localhost:8000` 回车即可。
4. 双击之后，会同时出现一个**黑色窗口**（里面有些英文日志）。**那是程序本体，不要关掉它**，最小化挂着就好。关掉黑窗口 = 停止服务。
5. 看界面**左下角**的状态：
   - 显示绿色「已连接 Ollama · qwen2.5:3b 就绪」→ 可以开聊了
   - 显示红色「未连接」→ 回去检查第 2 步模型是否装好、Ollama 是否在运行。
6. **第一次**发消息后，AI 可能要**等十几秒**才开口（模型正在加载进内存），这是正常的；之后聊天就很顺畅了。

> 关于端口：如果 `8000` 被别的软件占了，ChatSoul 会**自动改用 8001、8002……**，并在黑色窗口里告诉你实际地址，按那个地址打开就行，不用手动设置。

## 五、怎么玩

| 想做的事 | 怎么做 |
|----------|--------|
| 选一个角色聊天 | 看界面左侧角色列表，点一下任意角色即可开始；内置了墨尘先生、小喵、珂莱塔、老K 四个角色 |
| 导入你自己的角色 | 点左下角「＋」→「导入 skill 文件」，选一个 `.skill` 或 `.md` 角色文件，上传即生效 |
| 给角色换头像 | 点聊天界面顶部角色名左边的虚线圆圈，选一张图片（支持 png/jpg 等） |
| 删除不想要的角色 | 鼠标悬停左侧某个角色，右侧会出现「删除」按钮，确认后即删（含头像和记忆） |
| 清空当前角色记忆 | 点右上角「清空对话」，只清这一个角色 |
| 清空所有记忆 | 点左下角「＋」→「清空所有记忆」，一次性清掉全部本地对话缓存 |
| 换个浏览器 / 重开 | 直接再访问 `http://localhost:8000` 即可，对话都还在 |

## 六、常见问题

问：双击 ChatSoul.exe 黑窗口一闪就没了？
答：新版已经修了"端口冲突闪退"。如果还一闪而过，多半是上一次没关干净——先关掉那个黑色窗口（或在任务管理器结束 ChatSoul.exe），再双击一次。也可以双击同文件夹里的 `start.bat` 启动，出错会停在屏幕上方便你看原因。

问：界面左下角一直红色「未连接 Ollama」？
答：说明 AI 模型那一侧没准备好。请确认：① Ollama 已安装并在运行（第 2 步）；② 模型 `qwen2.5:3b` 已下载（用 `ollama list` 看得到）。两项 OK 后，关掉黑窗口重开 ChatSoul.exe 即可。

问：浏览器没自动打开？
答：手动打开浏览器，输入 `http://localhost:8000` 回车。如果提示打不开，确认 ChatSoul.exe 的黑窗口还在运行。

问：第一次回复特别慢 / 等了很久？
答：正常。AI 模型约 2–4GB，第一次要把它从硬盘加载进内存，需要十几秒；加载完之后每次回复都很快。

问：这个要联网吗？会泄露我的聊天内容吗？
答：**完全不需要联网**，所有对话都在你本机由 Ollama 生成，不上传任何服务器，私密性很好。只有**第一次下载模型**时需要联网。

问：占内存很大？
答：聊天时 Ollama 会把模型留在内存里（约 2–4GB），这是 AI 本身需要的。不聊天时可以在任务栏退出 Ollama 释放内存。你的对话记录是几 KB 的小文本，几乎不占空间。

## 七、隐私说明

- 你的每一句对话都只保存在**你这台电脑**的 `ChatSoul\data\` 文件夹里，不会上传到任何地方。
- 程序本身不收集任何信息，也不需要登录账号。
- 想要彻底删除所有聊天记录：界面里点「＋ → 清空所有记忆」即可；想连程序一起删，直接把这个 `ChatSoul` 文件夹扔进回收站就行。
